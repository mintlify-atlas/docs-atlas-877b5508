## What is Brautcloud?

Brautcloud is a wedding photo sharing platform that enables guests to upload and share pictures they took at a wedding with the bride and groom. Instead of scattered photos across multiple devices and messaging apps, Brautcloud centralizes all wedding photos in one beautiful, organized platform.

<Note>
Brautcloud is currently in early development. The platform is actively being built and new features are being added regularly.
</Note>

## Key features

Brautcloud provides essential functionality for wedding photo sharing:

- **Guest photo uploads** - Guests can easily upload photos from their devices during and after the wedding
- **Secure storage** - Photos are stored securely using S3-compatible object storage (AiStore)
- **Event organization** - Create separate events for different weddings, each with its own photo collection
- **JWT authentication** - Secure access control ensures only authorized users can view and upload photos
- **Modern tech stack** - Built with Spring Boot 4.0 and Angular 21 for a fast, reliable experience

## Who is Brautcloud for?

Brautcloud is designed for:

- **Brides and grooms** who want a centralized place to collect all photos from their wedding guests
- **Wedding guests** who want to share their photos easily without email attachments or messaging apps
- **Wedding planners** who need to organize photos from multiple events
- **Developers** looking for a real-world example of a full-stack Java Spring Boot and Angular application

## Technology overview

<Info>
Brautcloud uses modern, production-ready technologies:

**Backend:**
- Java 25 with Spring Boot 4.0.2
- PostgreSQL database
- AiStore (S3-compatible object storage)
- JWT authentication with Spring Security

**Frontend:**
- Angular 21
- Tailwind CSS for styling
- RxJS for reactive programming
</Info>
