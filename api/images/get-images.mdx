## Endpoint

```
GET /api/image/{eventID}
```

Retrieves all images uploaded to a specific event.

## Path Parameters

<ParamField path="eventID" type="Long" required>
  The unique identifier of the event
</ParamField>

## Response

Returns an array of ImageResponse objects.

<ResponseField name="images" type="array">
  Array of image objects associated with the event
  
  <ResponseField name="id" type="Long">
    Unique identifier for the image
  </ResponseField>
  
  <ResponseField name="imageKey" type="string">
    The S3 key/filename of the image
  </ResponseField>
  
  <ResponseField name="isVisible" type="boolean">
    Visibility status of the image
  </ResponseField>
  
  <ResponseField name="eventId" type="Long">
    The ID of the associated event
  </ResponseField>
</ResponseField>

```python Python

images = response.json()

for image in images:
    print(f"Image {image['id']}: {image['imageKey']}")
```

## Implementation Details

### Data Retrieval

The endpoint:
1. Queries the database for all images associated with the specified event ID
2. Transforms each Image entity to an ImageResponse DTO
3. Returns the list of image responses

### Image Response Mapping

The ImageResponse DTO is created from the Image model using the `fromImage()` static method, which extracts:
- Image ID
- Image key (S3 filename)
- Visibility status
- Associated event ID
