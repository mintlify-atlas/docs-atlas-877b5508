<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-3xl font-bold text-gray-900 dark:text-white mb-4">Quick start</h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">Get Brautcloud running in minutes</p>
  
  <Steps>
    <Step title="Clone the repository">
      Start by cloning both the frontend and backend repositories to your local machine.
      
      ```bash
      git clone https://github.com/domenicwalther/brautcloud.git
      cd brautcloud
      ```
    </Step>
    
    <Step title="Set up your environment">
      Configure your environment variables using Doppler or a `.env` file. You'll need credentials for PostgreSQL and S3-compatible storage (AiStore).
      
      ```bash
      # Required environment variables
      AWS_ACCESS_KEY_ID=your_access_key
      AWS_SECRET_ACCESS_KEY=your_secret_key
      AWS_ENDPOINT=http://localhost:9000
      POSTGRES_URL=localhost
      POSTGRES_USER=myuser
      POSTGRES_PW=secret
      ```
      
      <Accordion title="View full configuration">
        Check out the [environment variables guide](/deployment/environment-variables) for a complete list of required and optional configuration options.
      </Accordion>
    </Step>
    
    <Step title="Start the services">
      Launch the backend and frontend in development mode.
      
      ```bash
      # Start Docker services (PostgreSQL + AiStore)
      docker compose up -d
      
      # Start backend and frontend
      just dev
      ```
      
      The frontend will be available at `http://localhost:4200` and the API at `http://localhost:8080`.
    </Step>
    
    <Step title="Create your first event">
      Register an account and create your first wedding event to start collecting photos from guests.
      
      <Note>Events are password-protected and can generate QR codes for easy guest access.</Note>
    </Step>
  </Steps>
</div>
